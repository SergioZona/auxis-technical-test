AI Engineer — Take Home Exercise

Short, focused engineering exercise
Context

Five PDF documents containing tax tables are uploaded. The data should be stored in a database and a GET endpoint should return the stored records.

Deliverables: service URL, README, and a C4 Components diagram (Mermaid).

Objectives

  • Evaluate backend design for endpoints, persistence, and API behavior.
  • Understand the development cycle and tools used.
  • Demonstrate a practical AI approach for extracting structured data from PDFs.

Deliverables

  • Service URL to consume the service (can use a free endpoint like Vercel or Ngrok).
  • README with setup, run instructions, and architecture notes.
  • C4 Components diagram in Mermaid format.

Evaluation criteria

  • Clear backend and API design.
  • Practical approach to PDF data extraction and normalization.
  • Reasoning about parallel processing and bottleneck mitigation.
  • Documentation of development steps and tool choices.
